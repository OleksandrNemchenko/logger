Detailed Description

template<bool _ThrSafe, typename _TLogData>
class Logger::CLoggerBaseThrSafety< _ThrSafe, _TLogData >

Base class that implements different thread security strategies.

Logger::CLoggerBaseThrSafety class implements different thread security strategies depend on _ThrSafe template parameter. If it is true, this class will use std::mutex. Otherwise it will not use any thread safety primitives.

_TLogData parameter is used to declare OutData pure virtual function that is called from OutDataThrSafe call.

Template Parameters
_ThrSafeIf true, OutDataThrSafe will use std::mutex for multithread security. Otherwise it does not use any thread safety primitives.
_TLogDataLogger data type. It is used to declare OutData pure virtual function that will output logger data.

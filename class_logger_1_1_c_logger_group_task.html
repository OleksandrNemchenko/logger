Detailed Description

template<typename... _TTaskPtr>
class Logger::CLoggerGroupTask< _TTaskPtr >

Logger group task

You have to call Logger::CLoggerGroup::AddTask to create new instance and don't need to do it manually.

Warning
All loggers inside one task must have the same TLogData logger message data type.

Detailed Description

template<typename... _TTaskPtr>
class ALogger::ALoggerGroupTask< _TTaskPtr >

ALogger group task

You have to call ALogger::ALoggerGroup::addTask to create new instance and don't need to do it manually.

Warning
All loggers inside one task must have the same TLogData logger message data type.
